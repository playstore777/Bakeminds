git revert -n {first 7 chars of commit hash or commit message} (here -n tells the git to not commit manually, we will do it on our, own, it just have to put them to staging area.)

HEAD is the node which points to the latest commit we did.

if we want to revert the latest commit, then simply we can do:
    git revert HEAD -m {message} or git revert -n HEAD -m {message} (this stops from auto-commit, just puts them to staging area.)
